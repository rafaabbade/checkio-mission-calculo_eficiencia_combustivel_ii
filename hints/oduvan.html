Como verificar se a eficiência está abaixo de um certo valor?

Use uma condição if para verificar se a eficiência é menor que 10.

Como fazer a saída no formato certo?

Para retornar a eficiência do combustível e a recomendação, use uma tupla: return (eficiencia, recomendacao)

Minha fórmula está certa, mas o número de casas decimais não.

Use a função round(valor, 2) para garantir que a eficiência do combustível tenha duas casas decimais.
